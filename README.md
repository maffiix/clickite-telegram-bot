# Telegram Bot Miner
> It is a simple miner bot based on its own currency
## How to run this bot?
1. Go to @BotFather via Telegram and type /newbot
2. Fill all neccessary fields.
3. After that you will see the token.
4. Download & unzip this project (via Code > Download ZIP)
5. Create file .env and write there ONLY ONE string: `BOT_TOKEN=<your_bot_token>`
6. Open terminal and go to your folder with the project.
7. Type `python main.py` for Windows or `python3 main.py` for MacOS/Linux and press ENTER
8. Access your bot via its username. Now your bot is ready!
